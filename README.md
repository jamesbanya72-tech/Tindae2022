# Agriculture Support System
**PROG103 – Principle of Structured Programming | Group 1 | DIT1201F**
## Description
A GUI-based Python application that provides crop yield estimation,
fertilizer recommendations, and crop information for Sierra Leone farmers.
## SDG Alignment
- SDG 2: Zero Hunger
- SDG 8: Decent Work and Economic Growth
- SDG 15: Life on Land
## Requirements
- Python 3.8 or higher
- tkinter (included with standard Python)
## How to Run
```
python agriculture_support_system.py
```
